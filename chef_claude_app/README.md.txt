# 🍝 Chef Claude - Recipe Generator

This project is a simple **React application** that allows users to add ingredients, keep track of them in a list, and then generate a suggested recipe once enough ingredients are added.  
It was built to demonstrate **React components, state management, and conditional rendering**.

---

## ✨ Features
- ➕ Add ingredients to a dynamic list
- 📋 Display the ingredients you have on hand
- 🍲 Generate a recommended recipe from Chef Claude (once at least 4 ingredients are added)
- ✅ Uses multiple React components (`Header`, `Main`, `IngredientsList`, `ClaudeRecipe`)

---

## 🛠️ Technologies
- [React](https://react.dev/) (Vite project setup)
- JavaScript (ES6+)
- HTML5 / CSS3
